Quick Start Commands
1. Install dependencies
npm install
2. Compile contract
npx hardhat compile
3. Run tests
npx hardhat test
4. Start local blockchain (new terminal)
npx hardhat node
5. Deploy to local network
npx hardhat run scripts/deploy.js --network localhost
6. Open this page & paste contract address
Open index.html in browser → paste contract address → Connect

Contract ABI — Key Functions
notarizeDocument(bytes32 _docHash, string _metadata)
Register a document hash on-chain. Emits DocumentNotarized event.
WRITE
verifyDocument(bytes32 _docHash)
Returns owner, timestamp, metadata, isRevoked, chainId.
VIEW
isNotarized(bytes32 _docHash)
Quick boolean check if a document is registered.
VIEW
revokeDocument(bytes32 _docHash)
Owner only. Permanently marks document as revoked.
WRITE
transferOwnership(bytes32 _docHash, address _newOwner)
Transfer document ownership to another address.
WRITE
